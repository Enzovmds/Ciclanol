O que é o projeto Ciclanol

O projeto Ciclanol criado por um grupo de alunos do ensino médio técnico com o objetivo de minimizar os impactos ambientais gerados pelo descarte de resíduos no solo, na água e na atmosfera. A partir dessa premissa, surgiu a ideia de fabricar um biocombustível a base de óleo de cozinha já utilizado, ou a base de frutas que não seriam mais consumidas, para a finalidade de usar em geradores de luz para ampliar as áreas iluminadas de regiões mais afastadas dos grandes centros urbanos, também podendo ser utilizado como biocombustível para máquinas agrícolas.
